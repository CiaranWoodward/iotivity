CTT Gui Maven build Requirements:
Java 1.8
Maven
To install maven run follow procedure from here: https://maven.apache.org/install.html

Run ctt_gui_build.sh to build and create executable.
If build is successfull it will create CTT folder with CTT GUI executable

**If ConformanceTestTool_Temp is renamed please change the <module>../ConformanceTestTool_Temp</module> according to new name
of file : oic.ctt.parent/pom.xml
**change the CONFORMANCE_DIR of ctt_gui_build.sh to proper directory.

to build CTT GUI run ctt_gui_build.sh from terminal.
